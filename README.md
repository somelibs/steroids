# Steroids

Steroids provides a set of helpers for APIs and service-oriented Rails apps.

## Usage
n/a

## Installation

```ruby
gem 'steroids', :git => "git@github.com:somelibs/steroids.git", :branch => "master"
```

## Development

```
  bundle config local.steroids /path/to/local/git/repository
```

## Support

Rails 7+
